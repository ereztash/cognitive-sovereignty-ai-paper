# Model Assumptions

The simulation is a formal model, not an empirical proof.

## Core assumptions

1. Offloading increases immediate performance.
2. Offloading without calibration reduces productive effort.
3. Reflection demand restores part of the effort lost to offloading.
4. Foundational knowledge protects against epistemic dependence.
5. Cognitive sovereignty grows through calibrated evaluation and reflective use.
6. Cognitive sovereignty decays when a user repeatedly delegates without monitoring.

## Key limitation

The model is intentionally simple. It should be treated as a hypothesis generator for empirical research.
